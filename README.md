# rn_music
    一个使用react-native 和 react-navigation 写的项目，（功能未全部完成）
# install
    npm intall （或者 cnpm install）
    react-native run-android
# 功能
    推荐歌单
    推荐新歌
    banner
    点击歌单查看歌单详情
    歌手页面
    点击歌手查看歌手单曲
    点击歌曲播放
